# 🤖 Telegram Bot - Полнофункциональный бот с экономикой

## 📋 Описание

Мощный Telegram-бот с полноценной экономической системой, магазином, инвентарём, работами, ежедневными наградами и гибкой админ-панелью.

## ✨ Основные возможности

### Для пользователей:
- 💰 **Профиль** - просмотр баланса, уровня, опыта и статистики
- 🏪 **Магазин** - покупка предметов с фото и описанием
- 🎒 **Инвентарь** - управление купленными предметами
- 💼 **Работа** - 6 видов работ с кулдауном 2 часа
- 🎁 **Ежедневная награда** - streak-система с увеличением награды
- 🏆 **Рейтинг** - топ-10 игроков по балансу
- 🎨 **Кастомные кнопки** - админ может создавать любые кнопки с текстом и фото

### Для администраторов:
- 📊 **Статистика** - полная статистика бота
- 📢 **Рассылка** - массовая рассылка сообщений
- 🛠 **Управление магазином** - добавление товаров с фото
- 🎨 **Создание кнопок** - конструктор кастомных кнопок прямо в боте
- 👤 **Управление пользователями** - бан/разбан, выдача баланса
- 📝 **Логи** - просмотр всех действий

## 🚀 Установка

### 1. Клонирование и установка зависимостей

```bash
# Установка зависимостей
pip install -r requirements.txt
```

### 2. Настройка

Откройте `config.py` и настройте:

```python
# Получите токен у @BotFather
API_TOKEN = 'YOUR_BOT_TOKEN_HERE'

# Добавьте свой Telegram ID (узнать: @userinfobot)
ADMIN_IDS = [
    123456789,  # Ваш ID
]
```

### 3. Запуск

```bash
python main.py
```

## 🗄️ База данных

Бот использует SQLite с оптимизациями:
- ✅ WAL-режим для параллельной работы
- ✅ Connection pooling
- ✅ Retry-логика при database locked
- ✅ Thread-safe соединения
- ✅ Автоматические коммиты

### Таблицы:

- **users** - пользователи (баланс, уровень, опыт, баны)
- **inventory** - инвентарь пользователей
- **shop_items** - товары магазина
- **custom_buttons** - кастомные кнопки
- **transactions** - история транзакций
- **achievements** - достижения
- **daily_rewards** - ежедневные награды
- **work_history** - история работ

## 🎮 Функционал

### Экономика

- **Стартовый баланс:** 100₽
- **Ежедневная награда:** 50₽ + 10₽ за каждый день streak (макс 500₽)
- **Работы:** 6 видов с разными наградами (20-50₽)
- **Кулдаун работы:** 2 часа

### Магазин

- Неограниченное количество товаров
- Фото для каждого товара
- Категории товаров
- Автоматическое добавление в инвентарь при покупке

### Кастомные кнопки

Админ может создать любую кнопку:
1. Задать текст кнопки (отображается в меню)
2. Задать текст ответа (что отправит бот)
3. Прикрепить фото (опционально)

Кнопка автоматически появится в главном меню у всех пользователей.

### Система уровней

- Опыт начисляется за активность
- Уровень повышается при достижении порога
- Формула: `100 * (1.5 ^ уровень)`

## 🛡️ Безопасность

- ✅ Проверка бана перед каждым действием
- ✅ Валидация прав администратора
- ✅ Защита от SQL-инъекций (параметризованные запросы)
- ✅ Retry-логика при ошибках БД
- ✅ Thread-safe операции

## 🆘 Решение проблем

### Database is locked

Все ошибки `database is locked` исправлены:
- Добавлен WAL-режим
- Увеличен timeout до 30 секунд
- Retry-логика с экспоненциальной задержкой
- Thread-safe соединения

### Бот не отвечает

1. Проверьте токен в `config.py`
2. Убедитесь что бот запущен: `python main.py`
3. Проверьте логи в консоли

### Не работают админские функции

1. Проверьте что ваш ID в `ADMIN_IDS` в `config.py`
2. Узнать свой ID: отправьте любое сообщение @userinfobot
